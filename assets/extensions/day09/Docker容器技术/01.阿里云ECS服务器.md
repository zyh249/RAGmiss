# 01.阿里云ECS服务器

## 一、阿里云ECS概述

### 1. 云计算概述

云计算是一种按使用量付费的模式，这种模式提供可用的、便捷的、按需的网络访问， 进入可配置的计算资源共享池（资源包括网络，服务器，存储，应用软件，服务），这些资源能够被快速提供，只需投入很少的管理工作或与服务供应商进行很少的交互。



可以理解为：<span style="color: rgb(216,57,49); background-color: inherit">把所有IT资源（包括网络，服务器硬件，存储，开发平台，应用软件，服务等）整合到一起，为用户提供按需分配。像水、电这种资源一样，用多少付多少钱。</span>

### 2. 什么是公有云

公有云通常指云提供商为第三方用户提供能够使用的云，可以通过Internet使用，价格低廉，方便用户使用，使用户能够访问和共享基本的计算机基础设备资源，包括但不限于硬件、存储和带宽等。

&#x20;

优点：

* 用户按量付费

* 随用随取

* 资源丰富

* 无需自行安装和维护

缺点：

* 有违安全法规

* 数据量大时，使用费用会快速增加

&#x20;

品牌：

* 阿里云

* 腾讯云

* 华为云

* 金山云

* 天翼云

### 3. 阿里云ECS概述

阿里云ECS（Elastic Compute Service），即<span style="color: rgb(216,57,49); background-color: inherit">弹性计算服务</span>，是一种云计算基础架构服务，提供半托管的计算资源池，用户可以在其中创建、运行和管理自己的应用程序。阿里云ECS不仅可以提供高负载的计算和数据处理能力，还能够为用户提供弹性和可靠的云计算服务，确保在快速增长的业务环境中，应用程序能够拓展，并且极大的提升了可靠性和可用性。

### 4. 阿里云ECS特点

* 根据用户需求快速创建实例，可以选择操作系统、硬件配置、网络和存储等。

* 支持按量付费和包年包月的计费方式，具有价格灵活性和成本可控性。

* 支持多种应用场景的自动化部署，可以自动构建部署和弹性扩展。

* 提供高可用性和稳定性，具有故障转移、数据备份和业务恢复能力。

* 配套完整云计算产品生态，提供丰富的扩展功能，包括云监控、弹性伸缩、负载均衡、弹性 IP、安全组等

* 管理更加智能化，支持云 API 和控制台、Web 界面，满足各种管理需求。
