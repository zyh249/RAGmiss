# Softmax 归一化计算教程

## 一、Softmax 是什么？

Softmax 归一化的作用是：

> 把一组任意分数，转换成一组概率，并且所有概率加起来等于 1。

例如模型输出 3 个类别的原始分数：

```text
[2, 1, 0]
```

这 3 个数还不是概率，因为：

```text
2 + 1 + 0 = 3
```

而概率要求：

```text
所有概率加起来 = 1
```

Softmax 可以把它变成：

```text
[0.665, 0.245, 0.090]
```

也就是：

| 类别 | 原始分数 | Softmax 概率 |
|---|---:|---:|
| 类别 A | 2 | 66.5% |
| 类别 B | 1 | 24.5% |
| 类别 C | 0 | 9.0% |

---

## 二、Softmax 公式

假设有一组分数：

```text
x1, x2, x3
```

Softmax 公式是：

```text
softmax(xi) = e^xi / (e^x1 + e^x2 + e^x3)
```

更通用地写：

```text
softmax(xi) = e^xi / Σe^xj
```

其中：

| 参数 | 含义 |
|---|---|
| `xi` | 当前要计算的那个分数 |
| `xj` | 所有分数中的每一个 |
| `e` | 自然常数，约等于 2.718 |
| `e^xi` | 对当前分数做指数运算 |
| `Σe^xj` | 所有分数指数结果的总和 |

通俗理解：

```text
每个分数先做 e 的指数运算；
再除以所有指数值的总和；
最终得到概率。
```

---

## 三、Softmax 计算步骤

Softmax 的计算步骤固定是：

```text
1. 每个分数先做 e 的指数运算
2. 把所有指数值加起来
3. 每个指数值除以总和
4. 得到一组概率
```

---

## 四、例子一：[2, 1, 0] 的 Softmax 计算

假设原始分数是：

```text
[2, 1, 0]
```

---

### 1. 第一步：分别计算指数

需要计算：

```text
e^2, e^1, e^0
```

近似值：

```text
e^2 ≈ 7.389
e^1 ≈ 2.718
e^0 = 1
```

整理成表格：

| 原始分数 | 指数结果 |
|---:|---:|
| 2 | 7.389 |
| 1 | 2.718 |
| 0 | 1 |

---

### 2. 第二步：计算指数总和

```text
7.389 + 2.718 + 1 = 11.107
```

所以：

```text
指数总和 = 11.107
```

---

### 3. 第三步：每个指数值除以总和

第一个值：

```text
7.389 / 11.107 ≈ 0.665
```

第二个值：

```text
2.718 / 11.107 ≈ 0.245
```

第三个值：

```text
1 / 11.107 ≈ 0.090
```

所以：

```text
softmax([2, 1, 0]) ≈ [0.665, 0.245, 0.090]
```

---

### 4. 检查是否归一化

```text
0.665 + 0.245 + 0.090 = 1.000
```

说明 Softmax 输出的是一组概率。

---

## 五、为什么要用 e 指数？

因为指数函数会放大分数之间的差距。

原始分数：

```text
[2, 1, 0]
```

看起来差距只是：

```text
2 比 1 大 1
1 比 0 大 1
```

但是经过指数后：

```text
e^2 = 7.389
e^1 = 2.718
e^0 = 1
```

最高分被明显放大。

所以 Softmax 的特点是：

```text
分数越大，转换后的概率越大；
分数之间的差距会被指数函数放大；
所有概率加起来等于 1。
```

## 六、例子二：负数也可以计算

假设原始分数是：

```text
[-1, 0, 1]
```

---

### 1. 计算指数

```text
e^-1 ≈ 0.368
e^0 = 1
e^1 ≈ 2.718
```

---

### 2. 计算指数总和

```text
0.368 + 1 + 2.718 = 4.086
```

---

### 3. 每个指数值除以总和

第一个值：

```text
0.368 / 4.086 ≈ 0.090
```

第二个值：

```text
1 / 4.086 ≈ 0.245
```

第三个值：

```text
2.718 / 4.086 ≈ 0.665
```

所以：

```text
softmax([-1, 0, 1]) ≈ [0.090, 0.245, 0.665]
```

可以看到，负数也可以正常参与 Softmax 计算。

---

## 七、Softmax 的结果怎么理解？

如果模型输出：

```text
[2, 1, 0]
```

经过 Softmax 后得到：

```text
[0.665, 0.245, 0.090]
```

可以理解为：

| 类别 | 原始分数 | Softmax 概率 | 解释 |
|---|---:|---:|---|
| 类别 A | 2 | 66.5% | 最可能 |
| 类别 B | 1 | 24.5% | 次可能 |
| 类别 C | 0 | 9.0% | 可能性最低 |

Softmax 不会改变原始分数的大小顺序。

原始分数最高的类别，Softmax 后概率也最高。

## 八、Python 代码实现

### 1. 基础写法

```python
import numpy as np

scores = np.array([2, 1, 0])

exp_scores = np.exp(scores)

softmax_scores = exp_scores / np.sum(exp_scores)

print(softmax_scores)
```

输出：

```text
[0.66524096 0.24472847 0.09003057]
```

---

### 2. 逐行解释

```python
scores = np.array([2, 1, 0])
```

准备原始分数。

```python
exp_scores = np.exp(scores)
```

对每个分数做指数运算。

```python
softmax_scores = exp_scores / np.sum(exp_scores)
```

每个指数值除以所有指数值的总和。

```python
print(softmax_scores)
```

输出 Softmax 概率。

## 九、Softmax 的稳定写法

实际代码中，更推荐下面这种写法：

```python
import numpy as np

scores = np.array([2, 1, 0])

scores = scores - np.max(scores)

exp_scores = np.exp(scores)

softmax_scores = exp_scores / np.sum(exp_scores)

print(softmax_scores)
```

---

### 为什么要减去最大值？

如果分数很大，例如：

```text
[1000, 999, 998]
```

直接计算：

```text
e^1000
```

容易造成数值溢出。

所以先减去最大值：

```text
[1000, 999, 998] - 1000 = [0, -1, -2]
```

这样就变成：

```text
e^0, e^-1, e^-2
```

计算更安全。

---

### 减去最大值会不会改变结果？

不会改变 Softmax 的结果。

例如：

```text
softmax([2, 1, 0])
```

和：

```text
softmax([0, -1, -2])
```

结果是一样的。

因为：

```text
[2, 1, 0] - 2 = [0, -1, -2]
```

Softmax 对整体平移不敏感。

也就是说：

```text
所有分数同时减去同一个数，Softmax 结果不变。
```

## 十、稳定版 Softmax 函数

```python
import numpy as np

def softmax(scores):
    scores = np.array(scores)
    scores = scores - np.max(scores)
    exp_scores = np.exp(scores)
    probabilities = exp_scores / np.sum(exp_scores)
    return probabilities


result = softmax([2, 1, 0])

print(result)
```

输出：

```text
[0.66524096 0.24472847 0.09003057]
```

## 十一、Softmax 和普通归一化的区别

普通归一化可能是：

```text
每个值 / 所有值的总和
```

例如：

```text
[2, 1, 0]
```

普通归一化：

```text
2 / 3 = 0.667
1 / 3 = 0.333
0 / 3 = 0
```

结果：

```text
[0.667, 0.333, 0]
```

而 Softmax 是先做指数：

```text
e^2, e^1, e^0
```

再归一化：

```text
[0.665, 0.245, 0.090]
```

区别：

| 对比项 | 普通归一化 | Softmax |
|---|---|---|
| 是否使用指数 | 不使用 | 使用 `e^x` |
| 是否能处理负数 | 不稳定 | 可以 |
| 是否放大差距 | 不明显 | 会放大高分优势 |
| 输出是否为概率 | 可以 | 可以 |
| 深度学习常用程度 | 较少 | 非常常用 |

## 十二、Softmax 常见应用场景

Softmax 常用于多分类任务。

例如：

```text
猫、狗、鸟 三分类
```

模型输出原始分数：

```text
[3.2, 1.5, 0.4]
```

经过 Softmax 后：

```text
[0.80, 0.15, 0.05]
```

表示：

```text
猫：80%
狗：15%
鸟：5%
```

模型最终会选择概率最高的类别：

```text
猫
```

## 十三、最小记忆版

Softmax 一句话：

> Softmax 是把一组原始分数转换成概率分布的方法，分数越大，概率越大，所有概率加起来等于 1。

公式：

```text
softmax(xi) = e^xi / Σe^xj
```

计算步骤：

```text
1. 每个分数先做 e 的指数运算
2. 把所有指数值加起来
3. 每个指数值除以总和
4. 得到一组概率
```

稳定写法：

```text
先减去最大值，再做 Softmax。
```

记忆口诀：

```text
先指数，再求和，最后相除变概率。
```
